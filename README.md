# Marketing Campaign Performance Dashboard

## Overview
This dashboard analyzes the effectiveness of bank marketing campaigns and customer subscription behavior using Power BI.

## KPIs
- Total Customers
- Successful Conversions
- Conversion Rate (%)
- Average Campaign Contacts

## Visualizations
- Campaign Outcome Distribution
- Campaign Success by Job
- Contact Method Performance
- Campaign Performance by Month

## Filters
- Job Filter
- Marital Status Filter
- Education Filter

## Tools Used
- Power BI
- Bank Marketing Dataset

## Key Insights
- Conversion Rate: 11.27%
- Total Customers: 41.18K
- Successful Conversions: 5K
- Cellular contact method generated higher engagement.
- Subscription rates varied across different job categories.
